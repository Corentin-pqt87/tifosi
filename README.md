# tifosi

## Information
La base de donnée SQL n'a pas était faite via MySQL mais via l'applicaion DBeaver, le fichier est un export complet comprenant les tables, le contenue des tables et les views.
Le fichier export est un dump générer DBeaver, mais les tables ont bien était créé manuellement via les outils disponible dans DBeaver 25.3.2
